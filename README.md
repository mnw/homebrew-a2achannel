# homebrew-a2achannel

Homebrew tap for [A2AChannel](https://github.com/mnw/A2AChannel) — typed handoffs between Claude Code agents.

## Install

```bash
brew tap mnw/a2achannel
brew install --cask a2achannel
```

## Upgrade

```bash
brew upgrade --cask a2achannel
```

## Uninstall

```bash
brew uninstall --cask a2achannel
brew untap mnw/a2achannel
```
